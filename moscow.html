La communication visuelle me fascine depuis que je suis jeune. Je suis étonné à chaque fois de voir le produit finale d’une oeuvre. Ce qui m'intrigues d’avantage c’est les moyens utilisé pour arriver au produit finale. J’ai donc développé un très grand intérêts pour le processus. J'ai depuis approfondi mes connaissances dans les diverses plateformes de réalisations de contenus.
Grâce aux cursus de Interactive Media Designer de l'eracom je pourrais acquérir les connaissances nécessaires pour entrer dans le cadre professionelle.
